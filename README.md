# 자료구조 및 알고리즘 공부 (Python)

## 1단계: 기초 자료구조

- [x] **배열 / 리스트** — Python `list`의 동작 원리, 시간복잡도
- [ ] **연결 리스트** — 단일, 이중, 원형 연결 리스트 직접 구현
- [ ] **스택** — LIFO 개념, 괄호 검사 등 활용 문제
- [ ] **큐 / 덱** — FIFO 개념, `collections.deque` 활용

## 2단계: 기초 알고리즘

- [ ] **시간복잡도 / 공간복잡도** — Big-O 표기법 이해
- [ ] **정렬** — 버블, 선택, 삽입, 병합, 퀵 정렬 구현 및 비교
- [ ] **탐색** — 선형 탐색, 이진 탐색
- [ ] **재귀** — 재귀 호출 원리, 팩토리얼, 피보나치

## 3단계: 트리 & 해시

- [ ] **이진 트리 / 이진 탐색 트리(BST)** — 삽입, 삭제, 순회(전위/중위/후위)
- [ ] **힙** — 최소힙, 최대힙, `heapq` 모듈
- [ ] **해시 테이블** — 해시 함수, 충돌 처리, Python `dict` 원리

## 4단계: 그래프

- [ ] **그래프 표현** — 인접 행렬, 인접 리스트
- [ ] **BFS / DFS** — 너비 우선, 깊이 우선 탐색
- [ ] **최단 경로** — 다익스트라, 벨만-포드
- [ ] **위상 정렬** — DAG에서의 순서 결정

## 5단계: 고급 알고리즘

- [ ] **동적 프로그래밍(DP)** — 메모이제이션, 타뷸레이션
- [ ] **그리디 알고리즘** — 탐욕적 선택, 활동 선택 문제
- [ ] **백트래킹** — N-Queen, 조합/순열
- [ ] **분할 정복** — 병합 정렬 심화, 큰 수 곱셈

## 6단계: 실전 문제 풀이

- [ ] **백준(BOJ)** 단계별 문제 풀기
- [ ] **프로그래머스** 단계별 문제 풀기
- [ ] 각 자료구조/알고리즘당 최소 5~10문제 풀이

## 공부 방법

1. **개념 이해** → 직접 구현 → 관련 문제 풀이 순서로 진행
2. 각 자료구조는 **직접 클래스로 구현**해보기
3. 각 주제별로 `.py` 파일을 만들어 코드 정리
